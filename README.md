# Air Passenger Traffic Forecasting at SFO  

Forecasting air passenger traffic at **San Francisco International Airport (SFO)** using **multivariate time series** built from throughput + weather features.

## Tech Stack
- PyTorch
- scikit-learn
- Multivariate time series forecasting

## Brief Details
- Engineered a multivariate dataset combining **throughput** and **weather** features.
- Built and tested baseline and advanced models.
- Benchmarked models to compare forecasting performance.
